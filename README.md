**Don't forget to register for this year**

## Start

### Requirements

This projet needs:
- Node.js and npm

### Build

We use `gulp` to build everything. You can run `npm run build` to build it.
You can also use `npm run watch` during development.

## Launch

Projet is generated in build directory.
